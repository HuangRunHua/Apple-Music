# Music 歌词滚动效果
自从iOS/iPadOS 13更新之后，Apple为Apple Music带来了全新的滚动歌词功能。专辑图作为背景，加上高斯模糊。高亮的当前句歌词，仿佛浮于水面，十分引人注目；而后面的歌词，就像在深水里，若隐若现。
